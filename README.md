# Car Price Predictor

## Overview

Car Price Predictor is a Machine Learning web application developed using Python, Flask, and Scikit-Learn. The application predicts the estimated selling price of a car based on user inputs such as company, model, year of purchase, fuel type, and kilometers driven.

## Features

* Predicts car prices using a Linear Regression model
* User-friendly web interface
* Built with Flask
* Dynamic car model selection
* Fast and simple prediction system

## Technologies Used

* Python
* Flask
* Pandas
* NumPy
* Scikit-Learn
* HTML
* CSS
* Bootstrap

## Project Structure

* app.py – Flask application
* templates/index.html – Frontend page
* static/css/style.css – Styling
* Cleaned_Car_data.csv – Dataset
* LinearRegressionModel.pkl – Trained model

## How to Run

1. Clone the repository
2. Install dependencies:
   pip install -r requirements.txt
3. Run:
   python app.py
4. Open:
   http://127.0.0.1:5000
